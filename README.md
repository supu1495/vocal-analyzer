# 🎤 vocal-analyzer

カラオケ歌唱者向けのAI音声分析システムです。カラオケ録音をアップロードするだけで、ピッチ精度・歌唱技法・リズム感などの詳細なフィードバックを得られます。

> ⚠️ このリポジトリは現在開発中です。

---

## ✨ 主要機能

- **AI音源分離** — Demucs v4によるカラオケ録音からのボーカル自動抽出
- **ピッチ精度分析** — Crepeを用いたcent単位の高精度ピッチ検出
- **歌唱技法の検出** — ビブラート・こぶし・フォール・しゃくり・ロングトーンを自動評価
- **グルーブ・リズム感の評価** — BPMに対するタイミングのズレをジャンルに合わせて評価
- **声域の可視化** — 最低音〜最高音の声域を計測・経時変化で追跡
- **成長の記録** — 複数セッションにわたる歌唱力の向上を追跡
- **おすすめ楽曲提案** — 声域・歌唱スタイルに合った楽曲・アーティストを提案
- **プライバシー保護** — 録音音声は一切保存せず、分析結果のみ保持

## 🔄 処理の流れ

1. **アップロード** — 音声ファイル（WAV / MP3 / M4A、最大50MB・最大10分）をアップロード
2. **音源分離** — Demucsがボーカル・ドラム・ベース・その他に分離
3. **ボーカル抽出** — 話し声を除去し、歌声のみを抽出
4. **ピッチ検出** — Crepeがボーカルトラックのフレームごとのピッチを解析
5. **技法分析** — ビブラート・こぶし・フォール・しゃくり・ロングトーンを検出
6. **リズム評価** — ジャンルを考慮したグルーブ感の評価
7. **結果生成** — 総合スコアと具体的な改善アドバイスを生成
8. **保存** — 分析結果（JSON）のみPostgreSQLに保存。音声ファイルは即時削除

---

## 📊 分析結果の内容

各セッションで以下の情報が生成されます：

- **ピッチ正確性スコア**（0〜100）
- **リズム・グルーブ感スコア**（0〜100）
- **歌唱技法レポート**（ビブラートの周波数・深さ、こぶしの回数、フォールの滑らかさなど）
- **声域**（検出された最低音〜最高音）
- **ピッチ推移グラフ**（時系列の可視化）
- **改善アドバイス**（具体的なフィードバック）

## 🚀 セットアップ

> 前提条件：Docker と Docker Compose がインストール済みであること

```bash
# 1. リポジトリをクローン
git clone https://github.com/<your-username>/vocal-analyzer.git
cd vocal-analyzer

# 2. 環境変数をコピー
```

## 🔌 API一覧

| メソッド | エンドポイント | 説明 |
|---|---|---|
| `POST` | `/api/v1/analysis/upload` | 音声をアップロードして分析を開始 |
| `GET` | `/api/v1/analysis/{id}` | 分析結果を取得 |
| `GET` | `/api/v1/user/statistics` | ユーザーの統計・進捗を取得 |
| `POST` | `/api/v1/user/voiceprint` | 声紋を登録（高精度モード） |

すべてのエンドポイントはJWT認証が必要です。

---

## 🔐 セキュリティとプライバシー

- **JWT認証** — すべてのAPIリクエストに有効なJWTトークンが必要
- **HTTPS必須** — すべての通信をSSL/TLSで暗号化
- **音声データを保存しない** — 録音は処理後に即時削除。分析結果のみ保存
- **検索エンジン非公開** — `robots.txt`と`noindex`メタタグで非インデックス設定
- **UUIDベースのURL** — ユーザーページは固有のUUIDでのみアクセス可能
